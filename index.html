🔧 Información del Proyecto

🖥️ Frontend

  • React 18 + Vite 5
  • Tailwind CSS 3 (tema oscuro)
  • Context API (WeatherProvider)
  • Diseño responsivo mobile-first
  • Puerto: localhost:5173

⚙️ Backend

  • Node.js API REST
  • Health checks habilitados
  • Puerto: localhost:3001
  • Integración WeatherAPI.com
  • Reinicio automático

🐍 Servicios Python

  • Monitor de emergencias
  • Generador de reportes diarios
  • Volumen compartido de logs
  • Orquestado con Docker Compose

🧩 Componentes React

  • Dashboard — layout principal
  • Header — navegación sticky
  • StatsOverview — estadísticas nacionales
  • ProvinceCard — tarjeta por provincia
  • AlertPanel — panel de alertas
  • ReportPanel — reportes
  • EmergencyBanner — banner de emergencia

🚀 Cómo Iniciar

▶ Pasos de inicio

  • Ejecutar iniciar.bat (doble clic) para levantar todos los servicios
  • O usar docker-compose up desde esta carpeta
  • Esperar a que backend esté listo en localhost:3001
  • Abrir el dashboard en localhost:5173
  • Configurar WEATHERAPI_KEY si es necesario
